void FwWidget::setVisibleTime ( int  msec) [inline]

Устанавливает время в миллисекундах, сколько окно должно отображаться на экране.

Parameters:
msecВремя в миллисекундах, которое окно должно находится на экране.
Note:
Если окно отображается на экране, по вызова функции, будет запущен таймер сокрытия окна.
Если необходимо, чтобы находлось на экране бесконечно долго, установите время равное 0.
