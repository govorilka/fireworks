bool FwPixmap::isBlendEnabled ( ) const

Возращает значение флага blend.

Если функци вернулf true, то при рисование картинки на экране (используя функции Canvas), цвета картинки будут смешиваться с цветами заднего фона. Если false. Картинка будет рисоваться по верх.

Note:
По умолчанию, значение флага blend равно true
